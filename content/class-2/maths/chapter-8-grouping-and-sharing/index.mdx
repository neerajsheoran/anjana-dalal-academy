## Counting Wheels

Ramya is standing near a road, watching vehicles go by. She wants to count all the wheels.

- A **bicycle** has **2 wheels**.
- An **autorickshaw** has **3 wheels**.
- A **car** has **4 wheels**.

If there are **4 bicycles**, she could count the wheels by adding:

```
2 + 2 + 2 + 2 = 8
```

She is adding **2** for **4 times**. This is the same as saying **4 times 2 is 8**.

We can write this using a special sign called the **multiplication sign** (x):

```
4 x 2 = 8
```

This is much faster than adding again and again!

## What is Multiplication?

**Multiplication** is a quick way to add the **same number many times**.

Instead of writing 3 + 3 + 3 + 3 + 3, we can write:

```
5 x 3 = 15
```

This means **5 groups of 3** or **5 times 3**.

### More Examples

| Situation | Addition | Multiplication |
|-----------|----------|---------------|
| 2 autorickshaws, 3 wheels each | 3 + 3 = 6 | 2 x 3 = 6 |
| 7 cars, 4 wheels each | 4 + 4 + 4 + 4 + 4 + 4 + 4 = 28 | 7 x 4 = 28 |
| 3 butterflies, 2 wings each | 2 + 2 + 2 = 6 | 3 x 2 = 6 |
| 2 octopuses, 8 legs each | 8 + 8 = 16 | 2 x 8 = 16 |
| 4 rows of soldiers, 10 in each | 10 + 10 + 10 + 10 = 40 | 4 x 10 = 40 |

> **Tip:** When you see the word "times" or "groups of", think of multiplication!

## Order Does Not Matter

Here is a wonderful secret about multiplication:

**4 groups of 3** gives the same answer as **3 groups of 4**.

```
4 x 3 = 12
3 x 4 = 12
```

Think of a flower seller. Ram makes **4 bouquets with 3 flowers each**. Gopal makes **3 bouquets with 4 flowers each**. Both use **12 flowers** in total!

Similarly:

```
4 x 5 = 20   and   5 x 4 = 20
6 x 4 = 24   and   4 x 6 = 24
```

> **Remember:** You can multiply numbers in any order and get the same answer!

## Multiplication in Real Life

- Bharti puts **4 buttons** on each shirt. She has **7 shirts**. Total buttons = 7 x 4 = **28 buttons**.
- There are **8 packets of bindis**, with **5 bindis** in each packet. Total bindis = 8 x 5 = **40 bindis**.
- Rita bought **6 pencils** at Rs 4 each. Total cost = 6 x 4 = **Rs 24**.
- **5 people** sit in each car. There are **8 cars**. Total people = 8 x 5 = **40 people**.

## Sharing Equally -- Introduction to Division

Two siblings want to share **8 gulab jamuns** equally.

They give **1 gulab jamun** to each person, one by one:
- 1 for you, 1 for me.
- 2 for you, 2 for me.
- 3 for you, 3 for me.
- 4 for you, 4 for me.

Now all 8 gulab jamuns are shared equally. Each person gets **4 gulab jamuns**.

This is called **division** -- splitting things into equal groups.

> **Think of it this way:** Multiplication puts equal groups together. Division breaks things into equal groups!

## Making Groups

Garima has **20 star beads** and wants to make bracelets using **5 beads** each.

She keeps making bracelets until all beads are used up:
- 1st bracelet: 5 beads used, 15 left.
- 2nd bracelet: 5 more used, 10 left.
- 3rd bracelet: 5 more used, 5 left.
- 4th bracelet: 5 more used, 0 left.

She made **4 bracelets** from 20 beads!

```
20 shared into groups of 5 = 4 groups
```

## Key Words to Remember

| Word | Meaning |
|------|---------|
| **Multiplication** | A quick way to add the same number many times |
| **Times (x)** | The sign used for multiplication |
| **Groups of** | Equal-sized sets of things |
| **Multiplication table** | A list showing the results of multiplying a number by 1, 2, 3, 4... up to 10 |
| **Division** | Splitting a number into equal groups |
| **Sharing equally** | Giving the same amount to each person or group |
| **Product** | The answer you get when you multiply two numbers |
| **Even number** | A number that can be split into 2 equal groups (ends in 0, 2, 4, 6, 8) |
