In everyday life, we often describe mathematical relationships in words: "Shabnam is 3 years older than Aftab" or "the perimeter of a square is 4 times its side." In this chapter, we learn how to write such relationships in a concise and powerful way using **letter-numbers** (also called variables) and **algebraic expressions**.

## The Notion of Letter-Numbers

### Using Letters to Represent Numbers

Shabnam is 3 years older than Aftab. Instead of writing the full phrase "Aftab's age" each time, we can use a letter:

Let **a** = Aftab's age, and **s** = Shabnam's age.

Then the relationship becomes:

**s = a + 3**

This is an **algebraic expression**. The letters **a** and **s** are called **letter-numbers** -- they represent numbers that can change.

| Aftab's age (a) | Expression for Shabnam's age | Shabnam's age (s) |
|:---:|:---:|:---:|
| 4 | 4 + 3 | 7 |
| 10 | 10 + 3 | 13 |
| 23 | 23 + 3 | 26 |

Given Shabnam's age, we can also find Aftab's age: **a = s - 3**

If Shabnam is 20 years old, then Aftab's age = 20 - 3 = **17 years**.

### Matchstick Patterns

Parthiv is making L-shaped matchstick patterns. Each L needs 2 matchsticks.

- 5 Ls need 5 x 2 = 10 matchsticks
- 7 Ls need 7 x 2 = 14 matchsticks
- 45 Ls need 45 x 2 = 90 matchsticks

The general pattern: **Number of matchsticks = 2 x n**, where **n** is the number of Ls.

This expression **2 x n** (written as **2n**) tells us how many matchsticks are needed for any number of Ls.

### Expressions with Multiple Letter-Numbers

Ketaki buys coconuts at Rs 35 each and jaggery at Rs 60 per kg to make laddus. If she buys **c** coconuts and **j** kg of jaggery:

**Total cost = c x 35 + j x 60 = 35c + 60j**

For 7 coconuts and 4 kg jaggery:
Total cost = 35 x 7 + 60 x 4 = 245 + 240 = **Rs 485**

> **Key Idea:** Algebraic expressions allow us to express general mathematical relationships concisely. Such expressions are often called **formulas**.

### Formulas for Perimeters

| Shape | Formula |
|-------|---------|
| Square with side **q** | Perimeter = **4q** |
| Equilateral triangle with side **a** | Perimeter = **3a** |
| Regular pentagon with side **p** | Perimeter = **5p** |
| Regular hexagon with side **h** | Perimeter = **6h** |

## Omission of the Multiplication Symbol

In algebra, we simplify the way we write multiplication:

- **4 x n** is written as **4n**
- **5 x m** is written as **5m**
- **c x 35** is written as **35c** (the number comes first)

The multiplication sign is simply dropped to make expressions shorter and easier to read.

**Example:** Find the value of **7k** when k = 4.
7k = 7 x 4 = **28**

**Example:** Find the value of **5m + 3** when m = 2.
5m + 3 = 5 x 2 + 3 = 10 + 3 = **13**

> **Common Mistakes to Avoid:**
> - If a = -4, then 10 - a = 10 - (-4) = **14**, not 6
> - If d = 6, then 3d = 3 x 6 = **18**, not 36
> - If s = 7, then 3s - 2 = 3 x 7 - 2 = 21 - 2 = **19**, not 15

## Simplification of Algebraic Expressions

### The Perimeter of a Rectangle

The perimeter of a rectangle with length **l** and breadth **b**:

p = l + b + l + b

Since we can add terms in any order:

p = l + l + b + b = **2l + 2b**

Both expressions l + b + l + b and 2l + 2b are **equal** -- they give the same value for any given l and b. We call 2l + 2b the **simplified form**.

### Like Terms and Unlike Terms

**Like terms** are terms that involve the **same** letter-numbers:
- 5c, 3c, and 10c are like terms (all involve c)
- 12n and -4n are like terms (both involve n)

**Unlike terms** have **different** letter-numbers:
- 18c and 11d are unlike terms

**Important:** Like terms can be combined into a single term. Unlike terms cannot.

**Example:** 5c + 3c + 10c = (5 + 3 + 10)c = **18c** (using the distributive property)

### Combining Like Terms with Area

A big rectangle is split into two smaller rectangles with heights both equal to **v**, and widths 4 and 3.

- Area of smaller rectangles: 4v + 3v
- Area of the big rectangle: (4 + 3) x v = 7v

So: **4v + 3v = 7v** -- this is a visual proof of combining like terms!

Similarly, for subtraction: **12n - 4n = (12 - 4)n = 8n**

### Simplifying Expressions with Brackets

**Example:** A shop rents chairs at Rs 40 each and tables at Rs 75 each. When returned, Rs 6 per chair and Rs 10 per table are refunded.

For **x** chairs and **y** tables:
- Amount paid initially: 40x + 75y
- Amount refunded: 6x + 10y
- Net amount = (40x + 75y) - (6x + 10y)

Simplifying step by step:
1. Open the bracket (preceded by a minus sign, so signs change): 40x + 75y - 6x - 10y
2. Group like terms: (40x - 6x) + (75y - 10y)
3. Combine: **(40 - 6)x + (75 - 10)y = 34x + 65y**

### Quiz Score Example

Charu's scores in three quiz rounds are: 7p - 3q, 8p - 4q, and 6p - 2q, where **p** is the score for a correct answer and **q** is the penalty for a wrong answer.

Her total score:
(7p - 3q) + (8p - 4q) + (6p - 2q)

= 7p + 8p + 6p - 3q - 4q - 2q

= **(7 + 8 + 6)p - (3 + 4 + 2)q**

= **21p - 9q**

If p = 4 and q = 1: Total = 21 x 4 - 9 x 1 = 84 - 9 = **75**

### Comparing Scores

Her friend Krishita scored **23p - 7q** in total. The difference:

23p - 7q - (21p - 9q) = 23p - 7q - 21p + 9q = **2p + 2q**

Since both p and q are positive, Krishita scored more than Charu in every case!

## Using the Distributive Property

The distributive property works for letter-numbers just as it does for regular numbers:

**Example:** Simplify 4(x + y) - y

= 4x + 4y - y = 4x + (4 - 1)y = **4x + 3y**

## Expressions Are Not Always Equal

Two expressions that look similar may not be equal. Always check by substituting values.

**Are 5u and 5 + u equal?**

| u | 5u | 5 + u |
|---|-----|-------|
| 2 | 10 | 7 |
| 5 | 25 | 10 |
| 8 | 40 | 13 |

Clearly, **5u and 5 + u are different expressions** -- they give different values for the same u.

**Are 10y - 3 and 10(y - 3) equal?**

| y | 10y - 3 | 10(y - 3) |
|---|---------|-----------|
| 0 | -3 | -30 |
| 2 | 17 | -10 |
| 7 | 67 | 40 |
| 10 | 97 | 70 |

No, they are not equal. 10y - 3 means "3 less than 10 times y", while 10(y - 3) means "10 times (3 less than y)".

## Summing up Grid Patterns

When numbers are arranged in a grid pattern, there are multiple ways to find their sum. For example, consider a grid with 3s and letter-numbers r and s:

```
  3   3   3   3
  r   s   r   s
  r   s   r   s
  3   3   3   3
```

**Row-wise:** (4 x 3) + (r + s) + (r + s) + (4 x 3)
**Grouping like terms:** (8 x 3) + (r + r) + (s + s)
**Using symmetry:** 2 x (4 x 3 + r + s)

All three simplify to: **2r + 2s + 24**

## Rules for Simplification

Here is a summary of the key rules:

1. **Like terms** can be combined: 5a + 3a = 8a
2. **Unlike terms** cannot be combined further: 3a + 4b stays as 3a + 4b
3. **Brackets with + sign outside:** terms keep their signs: +(3a - 2b) = 3a - 2b
4. **Brackets with - sign outside:** signs of all terms inside flip: -(3a - 2b) = -3a + 2b
5. **Distributive property:** 4(x + y) = 4x + 4y

## Key Words to Remember

| Word | Meaning |
|------|---------|
| **Letter-number (Variable)** | A letter that represents a number which can take different values |
| **Algebraic expression** | A mathematical expression containing letter-numbers, numbers, and operations |
| **Formula** | An algebraic expression that describes a general mathematical relationship |
| **Like terms** | Terms that involve the same letter-numbers (e.g., 5x and 3x) |
| **Unlike terms** | Terms that involve different letter-numbers (e.g., 5x and 3y) |
| **Simplification** | Combining like terms to write an expression in a shorter form |
| **Distributive property** | Multiplying a number by a sum equals the sum of individual products: a(b + c) = ab + ac |
