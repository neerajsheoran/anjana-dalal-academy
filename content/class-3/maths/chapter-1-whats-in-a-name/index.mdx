## Counting Without Numbers

A long time ago, two brothers named **Deba** and **Deep** took care of cows. They did not know how to count! So how did they make sure no cow was lost?

Their friend gave them a smart idea:

> **Tip:** Make a mark on the wall for each cow that goes out. When a cow comes back, cross out one mark. If all marks are crossed, all cows are home!

This is called **tally marking**. You can keep track of things without even knowing numbers!

```
Going out:   | | | | |  (5 marks = 5 cows)
Coming back: ╳ ╳ ╳ ╳ ╳  (all crossed = all cows are home!)
```

If 2 marks are left without being crossed, it means **2 cows are still missing**.

## Comparing Names by Length

Every name has **letters**. Some names are **long** and some are **short**.

| Name | Number of Letters |
|------|:-:|
| Ox | 2 |
| Dog | 3 |
| Goat | 4 |
| Tiger | 5 |
| Monkey | 6 |
| Elephant | 8 |

- **Elephant** has the most letters (8). It is the **longest** name.
- **Ox** has the fewest letters (2). It is the **shortest** name.

> **Try this:** Write your name. Count how many letters it has. Now count the letters in your friend's name. Whose name is longer?

## Number Names and Their Letters

Numbers also have names! We can count how many letters are in each number name.

| Number | Number Name | Letters |
|:------:|-------------|:-------:|
| 1 | One | 3 |
| 2 | Two | 3 |
| 3 | Three | 5 |
| 4 | Four | 4 |
| 5 | Five | 4 |
| 10 | Ten | 3 |
| 17 | Seventeen | 9 |
| 43 | Forty Three | 10 |
| 56 | Fifty Six | 8 |

For two-word number names, we count **all the letters in both words** (not the space).

> **Fun fact:** The place with the **longest name** in India is **Venkatanarasimharajuvaripeta** in Andhra Pradesh. The places with the **shortest names** are **Ib** in Odisha and **Od** in Gujarat!

## Grouping Things

We can sort things into **groups** based on what is **common** about them.

### Example 1: Household things

| Things that need electricity | Things that do not need electricity |
|------------------------------|-------------------------------------|
| TV | Chair |
| Fan | Bed |
| Lamp | Pillow |
| Clock | Blanket |

### Example 2: Animals

| Animals that eat food | Things that do not eat food |
|---|---|
| Dog, Cat, Cow | Chair, Ball, Book |

You can group the **same things** in **different ways**. For example, you can group animals by size, by colour, or by how many legs they have.

> **Remember:** When you make groups, every thing must go into one group or the other. Nothing should be left out!

## Collecting Information

You can look around your class and collect information. For example, look at how children comb their hair:

| Hair Style | How many children? |
|---|---|
| Two ponytails | |

| Short hair | |

This is called **collecting data**. You look, you count, and you write it down.

## Key Words to Remember

| Word | Meaning |
|------|---------|
| **Tally marks** | Small marks (lines) used to count things one by one |
| **Longest name** | The name with the most letters |
| **Shortest name** | The name with the fewest letters |
| **Number name** | The word we use to say a number (like "Five" for 5) |
| **Grouping** | Putting things together that are alike in some way |
| **Data** | Information we collect by looking and counting |
