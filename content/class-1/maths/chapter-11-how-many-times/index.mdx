## A Visit to the Amusement Park

A group of children went to an amusement park. They saw many rides. Let us count how many children can sit on each ride!

---

## Counting in Groups

Instead of counting one by one, we can count in **groups**. This is faster and easier!

### Horse Swing

The horse swing has seats for **2 children** each. There are **4 horses**.

2 + 2 + 2 + 2 = **8**

So, **4 times 2** is **8**. That means 8 children can ride the horse swing!

### Toy Train

The toy train has **3 bogies**. Each bogie can hold **3 children**.

3 + 3 + 3 = **9**

So, **3 times 3** is **9**. That means 9 children can ride the toy train!

### Big Wheel

The big wheel has **5 compartments**. Each compartment can hold **4 children**.

4 + 4 + 4 + 4 + 4 = **20**

So, **5 times 4** is **20**. That means 20 children can ride the big wheel!

## What Does "Times" Mean?

When we add the **same number again and again**, we say it a certain number of **times**.

- 2 + 2 + 2 = adding 2, **three times** = **6**
- 3 + 3 + 3 + 3 = adding 3, **four times** = **12**
- 5 + 5 = adding 5, **two times** = **10**

This is called **repeated addition** — adding the same number over and over.

## Jalebis for Everyone!

After the rides, the children wanted to eat **jalebis**! Each plate has **3 jalebis**. They ordered **6 plates**.

3 + 3 + 3 + 3 + 3 + 3 = **18**

**6 times 3 is 18.** So they had 18 jalebis in all!

## Travelling in a Bus

Look at a bus with **9 rows of seats**. Each row has **2 seats**.

2 + 2 + 2 + 2 + 2 + 2 + 2 + 2 + 2 = **18**

**9 times 2 is 18.** The bus can hold 18 people.

Some crates of water bottles are kept in the bus. Each crate has **5 bottles**. There are **4 crates**.

**4 times 5 is 20.** There are 20 bottles in all.

## Counting Things in Groups Around Us

Many things around us come in groups!

| Thing | How Many in Each Group | How Many Groups | Total |
|-------|----------------------|----------------|-------|
| Wheels on a bicycle | 2 | 3 bicycles | 2 + 2 + 2 = **6 wheels** |
| Legs on a chair | 4 | 2 chairs | 4 + 4 = **8 legs** |
| Fingers on a hand | 5 | 2 hands | 5 + 5 = **10 fingers** |
| Petals on a flower | 5 | 3 flowers | 5 + 5 + 5 = **15 petals** |
| Wheels on an auto | 3 | 4 autos | 3 + 3 + 3 + 3 = **12 wheels** |

## Key Words to Remember

| Word | Meaning |
|------|---------|
| **Repeated addition** | Adding the same number again and again |
| **Times** | How many groups of the same number we add (e.g., 3 times 4 means 4 + 4 + 4) |
| **Group** | A set of things that are together |
| **Total** | The answer we get after adding all the groups |
| **Each** | Every single one (e.g., each plate has 3 jalebis) |
