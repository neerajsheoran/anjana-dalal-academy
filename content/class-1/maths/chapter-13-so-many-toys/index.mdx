## Counting Toys

Look at a toy shop! There are so many different toys. Let us count them.

| Toy | How Many? |
|-----|----------|
| Elephants | 3 |
| Teddy Bears | 4 |
| Cars | 6 |
| Dolls | 5 |

Now we can **compare** the number of each type of toy.

- The number of **cars** (6) is **more than** the number of **dolls** (5).
- The number of **elephants** (3) is **less than** the number of **dolls** (5).
- The number of **teddy bears** (4) is **more than** the number of **elephants** (3).
- The number of **cars** (6) is **more than** the number of **teddy bears** (4).

## More Than, Less Than, Equal To

When we compare two groups, we use these words:

| Word | Meaning | Example |
|------|---------|---------|
| **More than** | A bigger number | 6 is more than 3 |
| **Less than** | A smaller number | 3 is less than 6 |
| **Equal to** | The same number | 4 is equal to 4 |

## Colourful Flowers in a Garden

Look at a garden full of colourful flowers!

| Flower Colour | How Many? |
|--------------|----------|
| Blue | 3 |
| Orange | 5 |
| Red | 7 |
| Purple | 4 |

Now let us answer:
- Which colour has the **most** flowers? **Red** (7 flowers).
- Which colour has the **least** flowers? **Blue** (3 flowers).
- Are there more red flowers than blue flowers? **Yes!** 7 is more than 3.
- Are there less orange flowers than purple flowers? **No!** 5 is more than 4.

## Collecting and Counting

We can collect information by counting things around us. This is called **data**.

**Example:** Count the number of letters in the names of your classmates.

| Letters in Name | Children's Names |
|----------------|-----------------|
| 3 letters | Ram, Uma |
| 4 letters | Riya, Arun, Sita |
| 5 letters | Priya, Meena, Rahul |
| More than 5 | Manisha, Devendra |

Now we can see:
- Most children have names with **4 or 5 letters**.
- Fewer children have names with **3 letters**.

> **Activity:** Count how many children in your class have 3 letters, 4 letters, and more than 4 letters in their first name!

## Comparing Groups

When we look at groups of things, we can always ask:

1. Which group has the **most**?
2. Which group has the **least** (fewest)?
3. Which groups have an **equal** number?
4. Is one group **more than** or **less than** another?

**Example:** In a fruit basket:
- Apples: **4**
- Bananas: **6**
- Oranges: **4**

The bananas are the **most**. The apples and oranges are **equal** — both have 4. The apples and oranges have **less than** the bananas.

## Organising Information

When we count many things, it is helpful to write them down neatly in a **table**. A table helps us see the information clearly.

**Example:** Favourite fruits of 10 children:

| Fruit | Number of Children |
|-------|--------------------|
| Mango | 4 |
| Apple | 2 |
| Banana | 3 |
| Orange | 1 |

From this table, we can quickly see:
- **Mango** is the favourite fruit of the most children.
- **Orange** is liked by the fewest children.
- More children like **banana** (3) than **apple** (2).

---

## True or False?

Using the flower data from above, we can check if sentences are **true** (right) or **false** (wrong).

- "Red flowers are more than blue flowers." — **True** (7 is more than 3).
- "Orange flowers are less than purple flowers." — **False** (5 is more than 4, not less).
- "Blue flowers are the fewest." — **True** (3 is the smallest number).

## Key Words to Remember

| Word | Meaning |
|------|---------|
| **Data** | Information that we collect by counting or observing |
| **More than** | A bigger number or amount |
| **Less than** | A smaller number or amount |
| **Equal to** | The same number or amount |
| **Most** | The biggest amount in a group — the winner |
| **Least** | The smallest amount in a group — the fewest |
| **Table** | A neat way to write down information using rows and columns |
| **Compare** | To look at two or more things and see how they are different |
