## Introduction

In earlier classes, you studied **linear equations in one variable** such as x + 1 = 0 and 2y + 3 = 0, which have a unique solution. In this chapter, you will extend this study to **linear equations in two variables**. You will learn how to write them, find their solutions, and understand that they have infinitely many solutions, each represented as a point on a straight line in the Cartesian plane.

---

## Linear Equations

A **linear equation in two variables** is any equation that can be written in the form:

**ax + by + c = 0**

where **a**, **b**, and **c** are real numbers, and **a** and **b** are not both zero.

> **Key idea:** The word "linear" means the highest power of each variable is 1. There are no terms like x^2, y^2, or xy.

### Examples

* 2x + 3y = 9.35 can be written as 2x + 3y - 9.35 = 0 (a = 2, b = 3, c = -9.35)
* x - y/5 - 10 = 0 (a = 1, b = -1/5, c = -10)
* -2x + 3y = 6 can be written as -2x + 3y - 6 = 0 (a = -2, b = 3, c = -6)
* x = 3y can be written as x - 3y + 0 = 0 (a = 1, b = -3, c = 0)
* 2x + 5y = 0 (a = 2, b = 5, c = 0)
* 3x + 2 = 0 can be written as 3x + 0.y + 2 = 0 (a = 3, b = 0, c = 2)
* y - 2 = 0 can be written as 0.x + y - 2 = 0 (a = 0, b = 1, c = -2)

> **Key idea:** A linear equation in one variable like ax + b = 0 is a special case of a linear equation in two variables: ax + 0.y + b = 0. Similarly, y = 2 can be written as 0.x + 1.y - 2 = 0.

---

## Solution of a Linear Equation in Two Variables

A **solution** of a linear equation in two variables is a pair of values (one for x and one for y) that satisfy the equation. It is written as an **ordered pair (x, y)**.

**Example:** Consider 2x + 3y = 12.

* (3, 2) is a solution because 2(3) + 3(2) = 6 + 6 = 12.
* (0, 4) is a solution because 2(0) + 3(4) = 0 + 12 = 12.
* (6, 0) is a solution because 2(6) + 3(0) = 12 + 0 = 12.
* (1, 4) is NOT a solution because 2(1) + 3(4) = 2 + 12 = 14, not 12.

> **Key idea:** A linear equation in two variables has **infinitely many solutions**. You can choose any value for x and find the corresponding value of y (or vice versa).

### How to find solutions

**Method:** Pick any value for one variable, substitute it into the equation, and solve for the other variable.

**Example:** Find four solutions of x + 2y = 6.

| Choose | Substitute | Solve | Solution |
| ------ | ---------- | ----- | -------- |
| x = 0  | 0 + 2y = 6 | y = 3 | (0, 3)   |
| y = 0  | x + 0 = 6  | x = 6 | (6, 0)   |
| x = 2  | 2 + 2y = 6 | y = 2 | (2, 2)   |
| y = 1  | x + 2 = 6  | x = 4 | (4, 1)   |

## Graph of a Linear Equation in Two Variables

Every linear equation in two variables can be represented as a **straight line** on the Cartesian plane. Each point on this line is a solution of the equation, and every solution of the equation is a point on the line.

### How to draw the graph

1. Find at least **two solutions** of the equation.
2. Plot these solutions as points on the Cartesian plane.
3. Join the points with a straight line and extend it in both directions.

**Example:** Draw the graph of x + y = 7.

Some solutions: (0, 7), (7, 0), (3, 4). Plot these and draw a straight line through them.

![Graph of the linear equation x + y = 7 plotted on the Cartesian plane with points (0,7), (7,0), and (3,4) marked](graph-linear-equation-x-plus-y-equals-7.png)

### Special cases

* The equation **x = a** (a constant) represents a vertical line parallel to the y-axis, at a distance of a units from it.
* The equation **y = b** (a constant) represents a horizontal line parallel to the x-axis, at a distance of b units from it.
* The equation **y = mx** (passing through the origin) is a line through the origin.

**Example:** x = 3 is a vertical line through the point (3, 0).

**Example:** y = -2 is a horizontal line through the point (0, -2).

![Graph showing x = 3 as a vertical line and y = -2 as a horizontal line on the Cartesian plane](lines-parallel-to-axes.png)

> **Key idea:** Every point on the graph of a linear equation is a solution of the equation. Conversely, every solution of the equation is a point on the graph.

---

## Equations of Lines Parallel to the Axes

* **Lines parallel to the x-axis** have the equation **y = k** for some constant k. Every point on such a line has the same y-coordinate.
* **Lines parallel to the y-axis** have the equation **x = k** for some constant k. Every point on such a line has the same x-coordinate.
* The **x-axis itself** has the equation y = 0.
* The **y-axis itself** has the equation x = 0.

---

## Summary

1. An equation of the form **ax + by + c = 0** (a, b, c are real numbers; a, b not both zero) is a **linear equation in two variables**.
2. A linear equation in two variables has **infinitely many solutions**.
3. The graph of a linear equation in two variables is a **straight line**.
4. Every point on the line is a solution and every solution is a point on the line.
5. The equation **y = mx** represents a line passing through the origin.
6. The equation **x = a** is a vertical line; **y = b** is a horizontal line.
