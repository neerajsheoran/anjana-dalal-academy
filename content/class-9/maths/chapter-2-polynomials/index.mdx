## Introduction

In earlier classes, you studied algebraic expressions, their addition, subtraction, multiplication, and division. You also learnt how to factorise some algebraic expressions using identities like (x + y)^2 = x^2 + 2xy + y^2 and x^2 - y^2 = (x + y)(x - y). In this chapter, you will study a special type of algebraic expression called a **polynomial**, along with important theorems and identities for factorisation.

---

## Polynomials in One Variable

A **variable** is a symbol (like x, y, z) that can take any real value. A **polynomial in one variable** x is an algebraic expression where every term has a **whole number** as the exponent of x.

The general form of a polynomial of degree n is:

**p(x) = a\_n x^n + a\_(n-1) x^(n-1) + ... + a\_1 x + a\_0**

where a\_0, a\_1, ..., a\_n are constants and a\_n is not equal to 0.

> **Key idea:** Expressions like x + 1/x, sqrt(x) + 3, or 3sqrt(y) + y^2 are NOT polynomials because the exponent of the variable is not a whole number in at least one term.

### Terminology

* **Terms:** The individual parts of a polynomial separated by + or - signs. For example, in -x^3 + 4x^2 + 7x - 2, the terms are -x^3, 4x^2, 7x, and -2.
* **Coefficient:** The numerical factor of each term. In 4x^2, the coefficient is 4. In -x^3, the coefficient of x^3 is -1.
* **Degree:** The highest power of the variable in the polynomial. The degree of 3x^7 - 4x^6 + x + 9 is **7**.
* The degree of a non-zero constant polynomial is **0**.
* The degree of the **zero polynomial** is not defined.

### Classification by number of terms

| Name          | Number of Terms | Examples                 |
| ------------- | --------------- | ------------------------ |
| **Monomial**  | 1               | 5x^3, -2, y              |
| **Binomial**  | 2               | x + 1, x^2 - x           |
| **Trinomial** | 3               | x^2 + x + 1, 2 + x - x^2 |

### Classification by degree

| Name          | Degree | General Form                   |
| ------------- | ------ | ------------------------------ |
| **Linear**    | 1      | ax + b (a not 0)               |
| **Quadratic** | 2      | ax^2 + bx + c (a not 0)        |
| **Cubic**     | 3      | ax^3 + bx^2 + cx + d (a not 0) |

## Zeroes of a Polynomial

If p(x) is a polynomial and c is a real number such that **p(c) = 0**, then c is called a **zero** of the polynomial p(x). Equivalently, c is a **root** of the polynomial equation p(x) = 0.

**Example:** For p(x) = x - 1, we have p(1) = 1 - 1 = 0. So 1 is a zero of x - 1.

**Example:** For p(x) = x^2 - 2x, both x = 0 and x = 2 are zeroes because p(0) = 0 and p(2) = 4 - 4 = 0.

### Finding zeroes of a linear polynomial

For the linear polynomial p(x) = ax + b (where a is not 0), the zero is found by solving ax + b = 0, giving **x = -b/a**.

> **Key idea:** A linear polynomial has exactly one zero. A non-zero constant polynomial has no zero. Every real number is a zero of the zero polynomial.

A polynomial can have more than one zero. The maximum number of zeroes a polynomial of degree n can have is n.

## Remainder Theorem

When we divide a polynomial p(x) by a linear polynomial (x - a), we get a quotient q(x) and a remainder r:

**p(x) = (x - a) q(x) + r**

The **Remainder Theorem** states:

> **Key idea:** If p(x) is any polynomial of degree greater than or equal to 1 and a is any real number, then when p(x) is divided by (x - a), the remainder is **p(a)**.

**Example:** Find the remainder when p(x) = x^3 - 2x^2 - 4x - 1 is divided by (x - 1).

By the Remainder Theorem, remainder = p(1) = 1 - 2 - 4 - 1 = -6.

**Example:** Find the remainder when p(x) = 2x^3 + 3x^2 - 5x + 7 is divided by (2x + 1), i.e., by (x + 1/2).

Remainder = p(-1/2) = 2(-1/8) + 3(1/4) - 5(-1/2) + 7 = -1/4 + 3/4 + 5/2 + 7 = 10.

## Factor Theorem

The **Factor Theorem** is a special case of the Remainder Theorem:

> **Key idea:** If p(x) is a polynomial of degree n > 1 and a is any real number, then **(x - a) is a factor of p(x) if and only if p(a) = 0**.

In other words:

* If p(a) = 0, then (x - a) is a factor of p(x).
* If (x - a) is a factor of p(x), then p(a) = 0.

**Example:** Is (x + 1) a factor of x^3 + x^2 + x + 1?

Check p(-1) = (-1)^3 + (-1)^2 + (-1) + 1 = -1 + 1 - 1 + 1 = 0. Yes, (x + 1) is a factor.

**Example:** Find the value of k if (x - 1) is a factor of 4x^3 + 3x^2 - 4x + k.

Since (x - 1) is a factor, p(1) = 0. So 4 + 3 - 4 + k = 0, giving k = -3.

## Factorisation of Polynomials

### Factorising quadratic polynomials by splitting the middle term

To factorise ax^2 + bx + c, find two numbers p and q such that:

* p + q = b
* p times q = a times c

Then rewrite bx as px + qx and factor by grouping.

**Example:** Factorise 6x^2 + 17x + 5.

We need two numbers with sum 17 and product 30 (= 6 times 5). The numbers are 2 and 15.

6x^2 + 17x + 5 = 6x^2 + 2x + 15x + 5 = 2x(3x + 1) + 5(3x + 1) = **(3x + 1)(2x + 5)**

**Example:** Factorise 12x^2 - 7x + 1.

Product = 12, sum = -7. Numbers: -3 and -4.

12x^2 - 7x + 1 = 12x^2 - 3x - 4x + 1 = 3x(4x - 1) - 1(4x - 1) = **(3x - 1)(4x - 1)**

### Factorising cubic polynomials

For cubic polynomials, first use the Factor Theorem to find one factor, then factorise the remaining quadratic.

**Example:** Factorise x^3 - 2x^2 - x + 2.

Try x = 1: p(1) = 1 - 2 - 1 + 2 = 0. So (x - 1) is a factor.

Dividing: x^3 - 2x^2 - x + 2 = (x - 1)(x^2 - x - 2) = (x - 1)(x - 2)(x + 1)

**Example:** Factorise x^3 + 13x^2 + 32x + 20.

Try x = -1: p(-1) = -1 + 13 - 32 + 20 = 0. So (x + 1) is a factor.

x^3 + 13x^2 + 32x + 20 = (x + 1)(x^2 + 12x + 20) = **(x + 1)(x + 2)(x + 10)**

## Algebraic Identities

### Standard identities (from earlier classes)

* **Identity I:** (x + y)^2 = x^2 + 2xy + y^2
* **Identity II:** (x - y)^2 = x^2 - 2xy + y^2
* **Identity III:** x^2 - y^2 = (x + y)(x - y)
* **Identity IV:** (x + a)(x + b) = x^2 + (a + b)x + ab

### New identities

* **Identity V:** (x + y + z)^2 = x^2 + y^2 + z^2 + 2xy + 2yz + 2zx
* **Identity VI:** (x + y)^3 = x^3 + y^3 + 3xy(x + y) = x^3 + 3x^2y + 3xy^2 + y^3
* **Identity VII:** (x - y)^3 = x^3 - y^3 - 3xy(x - y) = x^3 - 3x^2y + 3xy^2 - y^3
* **Identity VIII:** x^3 + y^3 + z^3 - 3xyz = (x + y + z)(x^2 + y^2 + z^2 - xy - yz - zx)

> **Key idea:** If x + y + z = 0, then x^3 + y^3 + z^3 = 3xyz. This is a direct consequence of Identity VIII.

### Using identities for expansion

**Example:** Expand (3a + 4b + 5c)^2.

Using Identity V: = 9a^2 + 16b^2 + 25c^2 + 24ab + 40bc + 30ac

**Example:** Expand (2x + 1)^3.

Using Identity VI: = 8x^3 + 1 + 6x(2x + 1) = 8x^3 + 12x^2 + 6x + 1

**Example:** Expand (2a - 3b)^3.

Using Identity VII: = 8a^3 - 27b^3 - 18ab(2a - 3b) = 8a^3 - 27b^3 - 36a^2b + 54ab^2

### Using identities for evaluation

**Example:** Evaluate (99)^3.

99^3 = (100 - 1)^3 = 100^3 - 1 - 3(100)(1)(100 - 1) = 1000000 - 1 - 29700 + 297 = **970299** -- wait, let us recalculate:

Actually: (100 - 1)^3 = 1000000 - 1 - 300(99) = 1000000 - 1 - 29700 = 970299

**Example:** Evaluate (102)^3.

102^3 = (100 + 2)^3 = 1000000 + 8 + 600(102) = 1000000 + 8 + 61200 = **1061208**

### Using identities for factorisation

**Example:** Factorise 8a^3 + b^3 + 12a^2b + 6ab^2.

\= (2a)^3 + b^3 + 3(2a)^2(b) + 3(2a)(b)^2 = **(2a + b)^3**

**Example:** Factorise 27y^3 + 125z^3.

Using x^3 + y^3 = (x + y)(x^2 - xy + y^2):

\= (3y)^3 + (5z)^3 = **(3y + 5z)(9y^2 - 15yz + 25z^2)**

**Example:** Factorise 27x^3 + y^3 + z^3 - 9xyz.

\= (3x)^3 + y^3 + z^3 - 3(3x)(y)(z) = **(3x + y + z)(9x^2 + y^2 + z^2 - 3xy - yz - 3xz)**

### Useful special case

If x + y + z = 0, then x^3 + y^3 + z^3 = 3xyz.

**Example:** Find (-12)^3 + 7^3 + 5^3.

Since -12 + 7 + 5 = 0, the answer = 3(-12)(7)(5) = **-1260**.

## Summary

1. A **polynomial** p(x) in one variable x has the form a\_n x^n + ... + a\_1 x + a\_0 where a\_n is not 0.
2. Polynomials are classified as **monomials** (1 term), **binomials** (2 terms), or **trinomials** (3 terms).
3. By degree: **linear** (degree 1), **quadratic** (degree 2), **cubic** (degree 3).
4. A real number a is a **zero** of p(x) if p(a) = 0.
5. Every linear polynomial has exactly one zero.
6. **Factor Theorem:** (x - a) is a factor of p(x) if and only if p(a) = 0.
7. Key identities: (x + y + z)^2, (x + y)^3, (x - y)^3, and x^3 + y^3 + z^3 - 3xyz.
