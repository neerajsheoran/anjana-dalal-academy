## Introduction

Look around you -- patterns are everywhere in nature and daily life. The petals of a sunflower, the cells of a honeycomb, and even your salary increments follow patterns.

Consider these real-life examples:

- **Salary pattern:** Reena gets a starting salary of Rs 8000 with an annual increment of Rs 500. Her salaries form the sequence: 8000, 8500, 9000, 9500, ...
- **Ladder rungs:** Rungs decrease by 2 cm from bottom to top: 45, 43, 41, 39, 37, ...
- **Money box:** Shakila puts Rs 100 on her daughter's first birthday, then increases by Rs 50 each year: 100, 150, 200, 250, ...

Notice that in each case, we get the next number by **adding a fixed number** to the previous one. Such sequences are called **Arithmetic Progressions (AP)**.

---

## What Is an Arithmetic Progression?

An **Arithmetic Progression (AP)** is a list of numbers in which each term is obtained by adding a **fixed number** to the preceding term (except the first term).

This fixed number is called the **common difference**, denoted by **d**.

### Examples of APs

| AP | First term (a) | Common difference (d) |
|----|----------------|----------------------|
| 1, 2, 3, 4, ... | 1 | 1 |
| 100, 70, 40, 10, ... | 100 | -30 |
| -3, -2, -1, 0, ... | -3 | 1 |
| 3, 3, 3, 3, ... | 3 | 0 |
| -1.0, -1.5, -2.0, -2.5, ... | -1.0 | -0.5 |

> **Key idea:** The common difference d can be **positive** (increasing AP), **negative** (decreasing AP), or **zero** (constant AP).

### General Form of an AP

**a, a + d, a + 2d, a + 3d, ...**

where **a** = first term and **d** = common difference.

### How to Check if a List Is an AP

Calculate the difference between consecutive terms. If all differences are equal, the list is an AP.

**Example:** Is 2, 4, 8, 16, ... an AP?
- 4 - 2 = 2
- 8 - 4 = 4
- 16 - 8 = 8

The differences are NOT equal (2, 4, 8). So this is **not** an AP. (It is actually a geometric progression.)

**Example:** Is -1.2, -3.2, -5.2, -7.2, ... an AP?
- -3.2 - (-1.2) = -2
- -5.2 - (-3.2) = -2
- -7.2 - (-5.2) = -2

All differences are -2. So **yes**, it is an AP with d = -2.

### Finite and Infinite APs

- A **finite AP** has a fixed number of terms: 45, 43, 41, ..., 31 (8 terms)
- An **infinite AP** goes on forever: 1, 2, 3, 4, ...

## The nth Term of an AP

This is one of the most important formulas in this chapter.

### The Formula

For an AP with first term **a** and common difference **d**, the **nth term** is:

**a_n = a + (n - 1)d**

### How It Works

- 1st term: a_1 = a = a + (1 - 1)d
- 2nd term: a_2 = a + d = a + (2 - 1)d
- 3rd term: a_3 = a + 2d = a + (3 - 1)d
- ...
- nth term: a_n = a + (n - 1)d

> **Key idea:** The nth term formula lets you find ANY term of an AP directly, without listing all the terms before it. You just need to know the first term and the common difference.

The nth term is also called the **general term** of the AP. If the AP has **m** terms in total, the last term is **a_m = l = a + (m - 1)d**.

---

## Worked Examples on the nth Term

### Example 1: Find the 10th term of 2, 7, 12, ...

Here a = 2, d = 7 - 2 = 5, n = 10.

a_10 = 2 + (10 - 1)(5) = 2 + 45 = **47**

### Example 2: Which term of 21, 18, 15, ... is -81?

Here a = 21, d = -3, a_n = -81.

-81 = 21 + (n - 1)(-3)
-81 = 24 - 3n
-105 = -3n
**n = 35**

So -81 is the **35th term**.

### Example 3: Is 0 a term of this AP?

Using the same AP: 21 + (n - 1)(-3) = 0
3(n - 1) = 21
n - 1 = 7
**n = 8**

Yes, 0 is the **8th term**.

### Example 4: Determine the AP whose 3rd term is 5 and 7th term is 9

We have two equations:
- a + 2d = 5 ... (1)
- a + 6d = 9 ... (2)

Subtracting (1) from (2): 4d = 4, so **d = 1**
Substituting: a + 2 = 5, so **a = 3**

The AP is **3, 4, 5, 6, 7, ...**

### Example 5: Check whether 301 is a term of 5, 11, 17, 23, ...

a = 5, d = 6. If 301 is the nth term:

301 = 5 + (n - 1)(6) = 6n - 1
n = 302/6 = 50.33...

Since n must be a **positive integer**, 301 is **not** a term of this AP.

### Example 6: How many two-digit numbers are divisible by 3?

The two-digit multiples of 3 are: 12, 15, 18, ..., 99

This is an AP with a = 12, d = 3, a_n = 99.

99 = 12 + (n - 1)(3)
87 = 3(n - 1)
n - 1 = 29
**n = 30**

There are **30** two-digit numbers divisible by 3.

## Sum of First n Terms of an AP

Young Gauss was asked to find 1 + 2 + 3 + ... + 100. He paired up the numbers cleverly:

- S = 1 + 2 + 3 + ... + 100
- S = 100 + 99 + 98 + ... + 1

Adding: 2S = 101 + 101 + ... + 101 (100 times) = 10100

So S = **5050**.

### The Formulas

For an AP with first term **a**, common difference **d**, and **n** terms:

**Formula 1:** S_n = n/2 [2a + (n - 1)d]

**Formula 2:** S_n = n/2 (a + l)

where **l** is the last term.

> **Key idea:** Use Formula 1 when you know a, d, and n. Use Formula 2 when you know the first and last terms. Both give the same answer.

### Special Case: Sum of First n Natural Numbers

1 + 2 + 3 + ... + n = **n(n + 1)/2**

---

## Worked Examples on Sum

### Example 1: Find the sum of first 22 terms of 8, 3, -2, ...

a = 8, d = -5, n = 22

S_22 = 22/2 [2(8) + (22 - 1)(-5)]
= 11 [16 + 21(-5)]
= 11 [16 - 105]
= 11(-89)
= **-979**

### Example 2: Sum of first 14 terms is 1050, first term is 10. Find the 20th term.

S_14 = 1050, a = 10, n = 14.

1050 = 14/2 [2(10) + 13d]
1050 = 7[20 + 13d]
150 = 20 + 13d
130 = 13d
**d = 10**

a_20 = 10 + 19(10) = **200**

### Example 3: How many terms of 24, 21, 18, ... give a sum of 78?

a = 24, d = -3, S_n = 78

78 = n/2 [48 + (n - 1)(-3)]
78 = n/2 [51 - 3n]
156 = 51n - 3n^2
3n^2 - 51n + 156 = 0
n^2 - 17n + 52 = 0
(n - 4)(n - 13) = 0
**n = 4 or n = 13**

Both answers are valid! The sum of the first 4 terms equals the sum of the first 13 terms (= 78), because terms from the 5th to the 13th add up to zero.

### Example 4: TV Set Production Problem

A manufacturer produced 600 TV sets in the 3rd year and 700 in the 7th year. Production increases uniformly each year.

**Finding a and d:**
- a + 2d = 600 ... (1)
- a + 6d = 700 ... (2)

From (2) - (1): 4d = 100, so **d = 25**
Then a = 600 - 50 = **550**

**Production in 10th year:** a_10 = 550 + 9(25) = **775 sets**

**Total production in 7 years:**
S_7 = 7/2 [2(550) + 6(25)] = 7/2 [1100 + 150] = 7/2 (1250) = **4375 sets**

## Important Relationship

The **nth term** can also be found from the sum formula:

**a_n = S_n - S_(n-1)**

This is useful when the sum formula is given directly.

**Example:** If S_n = 4n - n^2, find the nth term.

S_n = 4n - n^2
S_(n-1) = 4(n-1) - (n-1)^2 = 4n - 4 - n^2 + 2n - 1 = 6n - 5 - n^2

a_n = S_n - S_(n-1) = (4n - n^2) - (6n - 5 - n^2) = 4n - n^2 - 6n + 5 + n^2 = **5 - 2n**

## Applications of AP in Daily Life

### Construction Penalty Problem

A contractor faces a penalty of Rs 200 for the first day of delay, Rs 250 for the second, Rs 300 for the third, and so on (increasing by Rs 50 each day). Total penalty for 30 days delay:

a = 200, d = 50, n = 30

S_30 = 30/2 [2(200) + 29(50)] = 15[400 + 1450] = 15(1850) = **Rs 27,750**

### Cash Prizes Problem

Seven prizes totalling Rs 700, each prize Rs 20 less than the preceding one. Find each prize.

Let the first prize = a. Then d = -20, n = 7, S_7 = 700.

700 = 7/2 [2a + 6(-20)]
200 = 2a - 120
2a = 320, so **a = 160**

The prizes are: **Rs 160, 140, 120, 100, 80, 60, 40**

### Tree Planting Problem

Each section of Class 1 to 12 plants as many trees as their class number. With 3 sections per class:

Total trees = 3 x (1 + 2 + 3 + ... + 12) = 3 x 12(13)/2 = 3 x 78 = **234 trees**

## Arithmetic Mean

If three numbers a, b, c are in AP, then **b** is the **arithmetic mean** of a and c:

**b = (a + c)/2**

For example, the arithmetic mean of 4 and 10 is (4 + 10)/2 = 7. And indeed, 4, 7, 10 form an AP.
