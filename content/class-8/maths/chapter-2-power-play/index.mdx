## 2.1 Experiencing the Power Play

### An Impossible Venture: Paper Folding

Take a sheet of paper and try folding it in half, again and again. Most people cannot fold a sheet of paper more than **7 times**. But here is the amazing part: if you could fold a paper **46 times**, it would be thick enough to reach the Moon!

How is this possible? Each fold **doubles** the thickness. Starting with a paper that is 0.001 cm thick:

| Folds | Thickness | Folds | Thickness |
|---|---|---|---|
| 1 | 0.002 cm | 10 | 1.024 cm |
| 2 | 0.004 cm | 17 | about 131 cm (4 feet) |
| 3 | 0.008 cm | 20 | about 10.4 m |
| 5 | 0.032 cm | 26 | about 670 m |
| 7 | 0.128 cm | 30 | about 10.7 km |

After just 30 folds, the thickness is about **10.7 km** -- the height at which aeroplanes fly! After 46 folds, the thickness exceeds **7,00,000 km**, which is more than enough to reach the Moon (about 3,84,400 km away).

> **Key idea:** This is the power of **exponential growth** -- when something doubles repeatedly, it grows incredibly fast. Each set of 10 folds multiplies the thickness by 1024 times!

## 2.2 Exponential Notation and Operations

### Writing Powers

When we fold paper, the thickness doubles each time. After n folds, the thickness becomes:

**0.001 cm x 2 x 2 x 2 x ... (n times) = 0.001 x 2^n cm**

We use a shorthand called **exponential notation**:

- n x n = **n²** (n squared, or n raised to the power 2)
- n x n x n = **n³** (n cubed, or n raised to the power 3)
- n x n x n x n = **n⁴** (n raised to the power 4)
- In general, n multiplied by itself **a** times = **n^a**

In the expression **5⁴ = 625**:
- **5** is the **base**
- **4** is the **exponent** (or power)
- **5⁴** is the **exponential form** of 625

### Examples

- 4 x 4 x 4 = 4³ = 64
- (-4) x (-4) x (-4) = (-4)³ = -64
- a x a x a x b x b = a³b² (read as "a cubed b squared")
- 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 = 2¹⁰ = 1024

### Prime Factorisation in Exponential Form

**Example:** Express 32400 as a product of prime factors in exponential form.

32400 = 2 x 2 x 2 x 2 x 3 x 3 x 3 x 3 x 5 x 5 = **2⁴ x 3⁴ x 5²**

## Laws of Exponents

### Law 1: Multiplying Powers with the Same Base

When multiplying powers with the same base, **add the exponents**:

**n^a x n^b = n^(a+b)**

**Example:** 3⁴ x 3³ = 3⁷ = 2187

This works because:
- 3⁴ x 3³ = (3 x 3 x 3 x 3) x (3 x 3 x 3) = 3⁷

### Law 2: Power of a Power

When raising a power to another power, **multiply the exponents**:

**(n^a)^b = n^(a x b)**

**Example:**
- 4⁶ = (4³)² = 64² = 4096
- 4⁶ = (4²)³ = 16³ = 4096
- 2¹⁰ = (2²)⁵ = (2⁵)² = 32² = 1024

### Law 3: Multiplying Powers with the Same Exponent

When multiplying different bases with the same exponent, **combine the bases**:

**m^a x n^a = (m x n)^a**

**Example:**
- 2⁵ x 5⁵ = (2 x 5)⁵ = 10⁵ = 1,00,000
- 3⁴ x 2⁴ = (3 x 2)⁴ = 6⁴ = 1296

### Law 4: Dividing Powers with the Same Base

When dividing powers with the same base, **subtract the exponents**:

**n^a / n^b = n^(a-b)** (where n is not 0)

**Example:**
- 2¹⁰⁰ / 2²⁵ = 2⁷⁵
- 2⁴ / 2³ = 2¹ = 2

## Zero and Negative Exponents

### Zero as an Exponent

Using the division law, if we divide a power by itself:

2⁴ / 2⁴ = 2⁰

But we also know that any number divided by itself is 1. Therefore:

**n⁰ = 1** (for any non-zero number n)

> **Key idea:** Any non-zero number raised to the power 0 equals 1. So 5⁰ = 1, 100⁰ = 1, and (-7)⁰ = 1.

### Negative Exponents

What happens when the exponent in the subtraction becomes negative?

If we halve a line of 2⁴ = 16 units five times:
- 2⁴ / 2⁵ = 2^(-1) = 1/2

This tells us:

**n^(-a) = 1/n^a**

**Examples:**
- 2^(-1) = 1/2
- 10^(-3) = 1/10³ = 1/1000
- 7^(-2) = 1/7² = 1/49

And conversely:
- n^a = 1/n^(-a)

> **Key idea:** A negative exponent means "take the reciprocal." So 10^(-3) means 1/1000, not a negative number.

---

## Power Lines

We can arrange all the powers of a number (positive, zero, and negative) along a number line. For powers of 4:

... 4^(-2) = 1/16 ... 4^(-1) = 1/4 ... 4⁰ = 1 ... 4¹ = 4 ... 4² = 16 ... 4³ = 64 ... 4⁴ = 256 ...

Moving right: each step **multiplies** by 4.
Moving left: each step **divides** by 4.

All the laws of exponents work for **any integer exponents** (positive, negative, or zero):

- **n^a x n^b = n^(a+b)**
- **(n^a)^b = n^(a x b)**
- **n^a / n^b = n^(a-b)**
- **m^a x n^a = (mn)^a**

## 2.4 Powers of 10

### Expanded Form Using Powers of 10

We can write any number using powers of 10:

47561 = (4 x 10⁴) + (7 x 10³) + (5 x 10²) + (6 x 10¹) + (1 x 10⁰)

For decimals, we use negative powers of 10:

561.903 = (5 x 10²) + (6 x 10¹) + (1 x 10⁰) + (9 x 10^(-1)) + (0 x 10^(-2)) + (3 x 10^(-3))

### Scientific Notation (Standard Form)

Very large numbers are hard to read. For example:
- Distance from the Sun to the centre of the Milky Way: 30,00,00,00,00,00,00,00,00,000 m

**Scientific notation** writes any number as **x x 10^y**, where 1 ≤ x < 10.

**Examples:**
- 5900 = 5.9 x 10³
- 20800 = 2.08 x 10⁴
- 80,00,000 = 8 x 10⁶
- Population of Mumbai (2 crore) = 2 x 10⁷

> **Key idea:** In scientific notation, the **exponent** tells you the scale (how many digits the number has), which is often more important than the exact digits. Changing the exponent by 1 means a **10 times** change in value!

### Why Scientific Notation Matters

The population of Kohima is about 1,42,395.
- Writing it as **1.42 x 10⁵** tells us it is "about 1 lakh 42 thousand."
- Writing it as **1.4 x 10⁵** tells us it is "about 1 lakh 40 thousand."

The number of digits in the coefficient shows how precisely we know the number. For most real-world facts, an approximate value is good enough.

## Linear Growth vs. Exponential Growth

Imagine building a ladder to the Moon (3,84,400 km away) with steps 20 cm apart. You would need about **192 crore steps** (1.92 x 10⁹).

Compare this with paper folding: just **46 folds** of exponential doubling covers the same distance!

- **Linear growth** is additive: 20 + 20 + 20 + ... (192 crore times)
- **Exponential growth** is multiplicative: 0.001 x 2 x 2 x 2 x ... (46 times)

> **Key idea:** Linear growth adds the same amount each step. Exponential growth multiplies by the same factor each step. Exponential growth starts slow but eventually overtakes linear growth by enormous amounts.

## Summary

- **Exponential notation:** n^a means n multiplied by itself a times. Here n is the **base** and a is the **exponent**.
- **Laws of exponents:**
  - n^a x n^b = n^(a+b)
  - n^a / n^b = n^(a-b)
  - (n^a)^b = n^(a x b)
  - m^a x n^a = (mn)^a
- **n⁰ = 1** for any non-zero n.
- **n^(-a) = 1/n^a** -- negative exponents give reciprocals.
- **Scientific notation** writes numbers as x x 10^y (where 1 ≤ x < 10) for easier reading and comparison.
- **Exponential growth** (doubling, tripling, etc.) is far faster than **linear growth** (adding a fixed amount).
